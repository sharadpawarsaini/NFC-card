# NFCure - Emergency Medical Identity (MVP) - Premium UI

## Overview
NFCure is a cloud-based emergency medical identity system (MVP). This repo contains a working Flask + MongoDB app with:
- User signup/login
- Health profile editing
- QR code generation for emergency view
- Public emergency view (scan access)
- PDF upload (medical reports)
- Access logs
- Simple admin dashboard

Designed for a premium UI using Tailwind CDN.
